## Elements

| Component | Description |
|-----------|-------------|
| [Alert](/components/alert) | Contextual feedback message with optional icon, title, and description |
| [Badge](/components/badge) | Displays a small label for status, category, or count |
| [Button](/components/button) | Interactive element that triggers an action or navigates to a URL |
