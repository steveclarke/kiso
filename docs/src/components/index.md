## Layout

| Component | Description |
|-----------|-------------|
| [Card](/components/card) | Container for grouping related content with header, content, and footer sections |
| [Empty](/components/empty) | Centered placeholder for empty content areas |
| [Separator](/components/separator) | Visual divider between content sections |
| [Stats Card](/components/stats_card) | Dashboard metric card for displaying KPIs |
| [Table](/components/table) | Data table with semantic HTML elements |

## Elements

| Component | Description |
|-----------|-------------|
| [Alert](/components/alert) | Contextual feedback message with optional icon, title, and description |
| [Badge](/components/badge) | Displays a small label for status, category, or count |
| [Button](/components/button) | Interactive element that triggers an action or navigates to a URL |
| [Kbd](/components/kbd) | Displays a keyboard key or shortcut inline |
| [Pagination](/components/pagination) | Navigation for paged content |

## Navigation

| Component | Description |
|-----------|-------------|
| [Breadcrumb](/components/breadcrumb) | Navigation trail showing the current page's location in the hierarchy |
| [Command](/components/command) | Command menu for search and quick actions with filtering and keyboard navigation |
| [Dropdown Menu](/components/dropdown_menu) | Menu of actions or functions triggered by a button |

## Overlay

| Component | Description |
|-----------|-------------|
| [Popover](/components/popover) | Floating panel anchored to a trigger element for displaying rich content |

## Forms

| Component | Description |
|-----------|-------------|
| [Checkbox](/components/checkbox) | Toggle control for boolean choices in forms |
| [Combobox](/components/combobox) | Autocomplete input with a searchable dropdown of suggestions |
| [Field](/components/field) | Composable form field wrapper with label, description, and error support |
| [Input](/components/input) | Single-line text field |
| [Input Group](/components/input_group) | Wraps an input with inline prefix/suffix addons |
| [Radio Group](/components/radio_group) | Group of radio buttons for selecting one option from a set |
| [Select](/components/select) | Dropdown list of options triggered by a button |
| [Switch](/components/switch) | Binary toggle for on/off states, built on a native checkbox |
| [Textarea](/components/textarea) | Multi-line text field |
| [Toggle](/components/toggle) | Two-state button that can be toggled on or off |
| [Toggle Group](/components/toggle_group) | Group of toggle buttons with single or multiple selection |
