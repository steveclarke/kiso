# Plus Jakarta Sans

The Kiso logo wordmark uses **Plus Jakarta Sans Bold (700)**.

## Files

| File | Purpose |
|------|---------|
| `PlusJakartaSans-Bold.ttf` | Logo wordmark weight |
| `PlusJakartaSans-Regular.ttf` | Brand materials body text |
| `PlusJakartaSans-v2.7.1-full.zip` | Complete family (all 14 weights + italics) |

## Usage in logo

The SVGs in `brand/svg/` have the text already converted to paths, so they
render correctly everywhere without the font installed. These font files are
here for reference and for creating new brand materials.

To install locally: copy the `.ttf` files to `~/Library/Fonts/` (macOS) or
`~/.local/share/fonts/` (Linux).
